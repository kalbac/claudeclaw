Features

Everything you need in a personal AI assistant. Nothing you don't.

🧠

Persistent Memory

SQLite with FTS5 full-text search. Semantic and episodic memories with salience-weighted decay. Pin important facts. Search across all conversations.

⏰

Cron Scheduler

Autonomous background tasks. Morning briefings, email monitoring, memory consolidation. Create tasks via Telegram or CLI with cron expressions.

📊

Web Dashboard

Monitor everything: memory stats, scheduled tasks, token usage, conversation history, audit log. Real-time status with auto-refresh.

🎙️

Voice Messages

Send voice notes via Telegram. Groq Whisper transcribes them for free. The assistant processes transcribed text as a normal message.

📧

Email Integration

Check unread messages, search by sender or subject. Works with any IMAP provider — Yandex, Gmail, Outlook, and more.

💊

Heartbeat

Proactive health checks. Define a checklist in HEARTBEAT.md. The assistant periodically reviews it and only notifies you when something needs attention.

Architecture

Two processes. One assistant. Full control.

[Telegram] ←→ [Claude Code Channels Plugin] ←→ [Claude Code CLI]
                                                      ↕
                                              [Your Project Dir]
                                               ├── CLAUDE.md  (core rules)
                                               ├── AGENT.md   (personality)
                                               ├── USER.md    (about you)
                                               └── CLI tools  (memory, schedule, mail)
                                                      ↕
                                                 [SQLite DB]
                                                      ↕
                                         [Background: npm start]
                                              ├── Dashboard (Hono)
                                              ├── Scheduler (cron)
                                              └── Heartbeat
